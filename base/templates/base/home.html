My Background

My first exposure to coding was at 8, when I got my first computer. I tinkered with the developer console in games like Half-Life and CS 1.6, and made simple modification to game files to increase bot difficulty and customize character attributes.

It wasn't until much later, during my chemistry undergrad, that I learned how coding could be used for far more practical appplications, such as data analysis and visualization.

I made the decision to continue a second degree in Computer Science to solidify my programming knowledge so that I can transition into more software-oriented roles. My focus is general software development however I would like to learn more about machine learning and data science.

I recently built a compound interest calculator web application that lets users punch in their investment details and provide them with a theoretical model of their money can grow over time. This was my first project focusing on data visualization and I'm looking forward to doing more of that. For my next project I want to tackle another data analysis/visualization problem but making use of AI/LLM models.
